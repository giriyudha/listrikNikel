Hubung Singkat

Kondisi abnormal yang menyebabkan arus mengalir pada jalur dengan impedansi sangat rendah

Tipe Hubung Singkat:

  1. 3 fasa (3 phase fault)
  2. Antar fasa (line to line)
  3. 2 fasa ke tanah (double line to ground)
  4. 1 fasa ke tanah (line to ground)

Sebab:

Secara umum, penyebab hubung singkat adalah kontak antar fasa atau fasa dengan ground

  1. Kerusakan pada lapisan pelindung konduktor
  2. Penuaan isolasi
  3. Kerusakan jalur transmisi sehingga kabel terlepas dan bersentuhan dengan tanah
  4. Peralatan logam tertinggal pada panel sehingga bersentuhan dengan busbar
  5. Kegagalan isolasi
  6. Benda asing bersentuhan dengan 2 atau 3 busbar sehingga terbentuk jalur arus

Akibat:

Secara umum, akibat hubung singkat adalah kenaikan arus dan panas berlebih

  1. Aliran arus keluar dari konduktor
  2. Overheat pada peralatan
  3. Ledakan jika nilai arus diatas breaking capacity
  4. Fluktuasi tegangan yang dapat memperpendek umur peralatan
  5. Aliran daya padam

Menghitung Arus Hubung Singkat 3fasa

Konsep dasar:

I = V / Z atau KVA / V

Data yang dibutuhkan:
  • Daya (VA) sistem
  • Level tegangan
  • Impendansi konduktor atau peralatan
  • Titik terjadinya gangguan

Perhitungan dan Simulasi SC 3 fasa
Data:

MVA Grid (MVA)

kV Grid (kV)

kV Bus 1 (kV)

MVA Tr1 (MVA)

Z Tr1 (%)

Z Kabel (ohm)

MVA Tr2 (MVA)

Z Tr2 (%)

kV Bus 2 (kV)

Z Beban (ohm)

Hasil perhitungan:

1.Impendasi Base atau Zb1 pada level teg 0kV = kV^2 / MVA grid= 0^2 kV / 0 MVA = 0 Ohm

2.Impendasi Base atau Zb2 pada level teg 0kV = kV^2 / MVA grid= 0^2 kV / 0 MVA = 0 Ohm

3.Impendasi pu TR1 = (Z TR1 * kV^2 Bus 1 / MVA Tr1 ) / Zb1 = (0 % * 0^2 kV/ 0 MVA) / 0 Ohm = 0 pu

4.Impedansi pu Kabel = Z kabel / Zb = 0 ohm / 0 ohm = 0 pu

5.Impendasi pu TR2 = (Z TR2 * kV^2 Bus 2 / MVA Tr2 ) / Zb2 = (0 % * 0^2 kV/ 0 MVA) / 0 Ohm = 0 pu

6.Total impedansi pu paralel beban = (Z load / Zb2) /3 = 0ohm / 0 ohm = 0 pu

7.Impedansi jalur dari grid - titik SC = Z TR 1 + Z Kabel + Z TR2 = 0 + 0 + 0 = 0 pu

8.I short circuit dari grid = (MVA / 1,73 * KV Bus 2) * 1/Impendansi poin no 7 = (0/1,73* 0)* 1/0 = 0 kA

9.I short circuit dari beban = (MVA / 1,73 * KV Bus 2) * 1/Impendansi beban = (0/1,73* 0)* 1/0 = 0 kA

Inrush Trafo

Ketika sebuah transformator di beri tegangan (energized), arus yang nilainya bisa mencapai 3 - 60 kali dari arus FLA akan mengalir selama beberapa saat

Sebab:

  1. Saat trafo dinyalakan dengan sisi sekunder open, trafo menjadi beban induktif
  2. Pada awal dinyalakan, trafo belum memiliki fluks sehingga tidak bisa melakukan induksi ke sisi sekunder
  3. Untuk menghasilkan fluks, inti trafo akan tersaturasi dan membutuhkan arus yang tinggi

Akibat:

Secara umum, karena hanya terjadi dalam waktu yang sangat singkat, inrush tidak menimbulkan kerusakan permanen namun jika terdapat kegagalan isolasi, dapat terjadi arcing dan kerusakan peralatan
